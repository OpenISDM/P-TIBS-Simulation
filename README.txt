There are simple introduction of each code.

(1)One :
   
   Use The One Simulator to simulate the node(POS & mobile) connect and move, then make a eventlog file(save each device connect up and down time).

(2)TreatOneEvent :

   To read the eventlog file make by One, and then simulate the document transfer and tag produce and transfer.

(3)TIBSTraceBySim :

   Insert the TreatOneEvent produce tag, and then show the trace result to Gephi.
